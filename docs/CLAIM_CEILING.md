# Claim Ceiling

ProofLedger AI is a hackathon prototype.

Allowed claims:

- It stores synthetic AI-agent decision receipts.
- It uses DynamoDB as the evidence ledger when configured with AWS credentials.
- It shows HOLD, BLOCK, and HUMAN_REVIEW_ONLY outcomes.
- It demonstrates a public-safe evidence memory pattern.

Blocked claims:

- production approval
- financial authorization
- compliance certification
- security certification
- legal review
- field validation
- autonomous business action
- customer deployment readiness
