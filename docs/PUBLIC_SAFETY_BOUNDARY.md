# Public Safety Boundary

Allowed in the public repo:

- synthetic agent decision packets
- public-safe receipt schema
- simple validator
- DynamoDB schema
- Vercel dashboard
- tests and docs
- architecture diagram

Not allowed:

- real customer data
- AWS credentials
- tenant secrets
- private validation banks
- hidden thresholds
- verifier seeds
- private negative controls
- production approval keys
- private customer strategy
