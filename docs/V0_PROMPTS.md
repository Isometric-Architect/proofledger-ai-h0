# v0 Prompt Pack

Use these prompts in v0 to refine the dashboard.

## Prompt 1: Dashboard

Build a clean enterprise dashboard for "ProofLedger AI". It should show three status cards: HOLD, BLOCK, HUMAN_REVIEW_ONLY. Below the cards, show a receipt table with subject, agent, requested action, allowed action, findings, and receipt hash. Use a restrained B2B design. Do not add fake auth or payment flows.

## Prompt 2: Receipt Detail

Add a subject timeline page. It should show all receipts for one subject ID, sorted by time, with the decision, recommendation, findings, and receipt hash. Keep the strongest action as human review only.

## Prompt 3: Demo Copy

Make the UI language clear: "AI recommendation is not approval." "External action disabled." "Allowed action: open human review."

## Prompt 4: Redaction Boundary

Add a small footer that says: "Synthetic demo data only. No customer data or credentials."
